# vue项目配置文件
* 在 static 目录添加文件 ``config.js``
```code
const globalData = {
  a: 'hello'
}
```
* 在 index.html 中引入
```code
<script src="static/config.js"></script>
```
* 在 ``eslintrc.js`` 中添加忽略报错
```code
globals: {
    globalData: true
  },
```
* 在页面中使用 ``globalData.a`` 即可获取数据，打包后只需要修改 ``config.js`` 文件中 ``a`` 的内容即可
